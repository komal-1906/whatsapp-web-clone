📱 WhatsApp Website Clone

This project is a frontend clone of the official WhatsApp website, built using HTML and CSS.
The goal of this project is to practice recreating real-world website layouts and improve frontend development skills.


🚀 Features

-> Modern UI inspired by the official WhatsApp website

-> Multiple structured sections

-> Hover effects and UI interactions

-> CSS animations

-> Clean and organized layout


🛠️ Technologies Used

HTML5

CSS3

Flexbox

CSS Animations
